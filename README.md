# Student Performance Analysis Dashboard (Excel Project)

This project explores the key factors influencing student performance using Microsoft Excel. The goal is to understand how study hours, attendance, parental involvement, and resource access affect exam outcomes.

## Project Objectives
- Identify major factors that influence academic performance.
- Analyze trends across study hours, attendance, parental support, and resource access.
- Provide insights that can support better educational planning.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Interactive Slicers
- Dashboard Design

## Key Insights
- Students with high attendance and longer study hours performed significantly better.
- Parental involvement showed a strong positive impact on exam scores.
- Access to learning resources improved learning outcomes.

## Files in This Repository
